# Study Notes Summarizer

A simple tool that turns pasted lecture notes or articles into a short summary
and a set of quiz questions, using the Gemini API.

## Features
- Paste any text and get an instant AI-generated summary
- Auto-generated quiz questions to self-test
- Runs entirely in the browser — no backend required

## Tech stack
HTML, CSS, JavaScript, Google Gemini API

## How to run
1. Clone this repo
2. Open `index.html` in your browser (or use Live Server in VS Code)
3. Get a free API key at [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
4. Paste your key into the app and try it out
